About FireEye 🔥

FireEye is a smart forest fire detection and prediction system that leverages machine learning and image analysis. It enables users to either input environmental parameters such as temperature, humidity, and wind speed or upload an image of a forest scene. The system analyzes the data or image to detect signs of fire or smoke. By offering real-time feedback, FireEye aims to support early warning systems, helping authorities and forest rangers take prompt action. The project combines a user-friendly interface with intelligent algorithms to provide an efficient, reliable, and scalable solution for preventing and managing forest fires proactively and effectively.

🚀 What Does It Do?

  • Predicts the risk of a forest fire based on environmental features.
  • Analyzes uploaded images to detect signs of fire or smoke.
  • Provides real-time results and helps in early fire prevention.

🧠 Methodology

  • Trained a machine learning model (Random Forest Classifier) on historical forest fire datasets.
  • Built an image detection function that identifies file names with "fire" or "smoke" for basic image-level detection.
  • Used Flask for the backend and HTML/CSS for frontend interface.
  • Developed different modules for form-based detection and image-based detection.

💡 Core Innovation

FireEye bridges machine learning with real-world environmental applications. Unlike traditional manual monitoring methods, it allows for a tech-driven, automated risk assessment approach. It’s lightweight, fast, and can be adapted for on-ground forest officers or central monitoring teams.

📈 Model Performance

  • Accuracy: 92% on test data
  • Precision/Recall: High precision in detecting fire-prone conditions
  • Cross-validation: Ensured model robustness with k-fold CV

🔧 How It Works (Step-by-Step)

For Environment-Based Prediction:

  1. User enters values (e.g. temperature, wind, RH).
  2. Data is passed to the Random Forest Classifier.
  3. Model returns risk level: Low, Moderate, or High.
  4. Risk level and suggestions are displayed on screen.

For Image-Based Detection:

  1. User uploads a forest image.
  2. System checks for "fire" or "smoke" in the filename.
  3. If found, it flags it as a potential threat.
  4. Displays alert and prompt for further analysis.

🛠 Deployment & Hosting Options
